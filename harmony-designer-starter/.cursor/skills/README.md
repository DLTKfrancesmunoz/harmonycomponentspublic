# Cursor skills (Harmony designer starter)

These skills ship with the designer starter kit. **All skills live under `.cursor/skills/`** in this project.

| Folder | Purpose |
|--------|---------|
| `harmony/` | Harmony hub: components, ShellLayout, theme/mode |
| `harmony-usage-rules/` | Compliance and accessibility |
| `harmony-ux-principles/` | UX critique |
| `layout-builder/` | Compose layouts in React using `src/components/harmony/` |
| `design-patterns/` | Pattern reference docs, registry, `scripts/create_pattern.py`, `scripts/search_patterns.py` |

**Not included:** `harmony-converter` (this kit is React-only).

See also [`.cursor/DESIGNER_GUIDE.md`](../DESIGNER_GUIDE.md) for slash commands and workflows.
