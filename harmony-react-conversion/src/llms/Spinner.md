# Spinner

Loading spinner with size.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| size | 'sm' \| 'md' \| 'lg' | 'md' | sm: Small.; md: Medium.; lg: Large. |
| className | string | '' | Additional CSS classes applied to the root element. |

## Usage

```tsx
import { Spinner } from './components/harmony/Spinner';

<Spinner size="sm" />
```

## Variants

- **sm**: Small.
- **md**: Medium.
- **lg**: Large.

## CSS Classes

- `.spinner`
- `.spinner--sm`
- `.spinner--md`
- `.spinner--lg`

## Dependencies

None (standalone component).
