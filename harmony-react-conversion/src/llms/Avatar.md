# Avatar

User or entity thumbnail image with optional fallback and size presets.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| size | 'sm' \| 'md' \| 'lg' | 'md' | sm: Small.; md: Medium.; lg: Large. |
| className | string | '' | Additional CSS classes applied to the root element. |

## Usage

```tsx
import { Avatar } from './components/harmony/Avatar';

<Avatar size="sm" />
```

## Variants

- **sm**: Small.
- **md**: Medium.
- **lg**: Large.

## CSS Classes

- `.avatar`
- `.avatar--sm`
- `.avatar--md`
- `.avatar--lg`

## Dependencies

- Icon
