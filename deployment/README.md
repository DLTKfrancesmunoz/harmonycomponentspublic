# Deployment metadata

This directory holds optional metadata for the public Harmony distribution (`DLTKfrancesmunoz/harmonycomponentspublic`).

There is no bundled CI or package registry configuration in this repository. Install the design system from Git with `npm install git+https://github.com/DLTKfrancesmunoz/harmonycomponentspublic.git` (or a path/file dependency), and run builds locally with `npm run build`.
